## Cowrie Collection

A collection to defend [Cowrie](http://github.com/cowrie/cowrie) against common attacks:
 - Cowrie parser
 - Cowrie bruteforce attacks

## acquis.yaml

Example acquisition for this collection:

```
---
filenames:
 - /home/user/cowrie/var/log/cowrie/cowrie.log
labels:
  type: cowrie
```
